<task type="auto">
  <name>Task 2: Configure Vite 8 + Tailwind CSS 4 + Alpine.js build pipeline</name>
  <files>
    package.json, vite.config.js, src/css/main.css, src/js/main.js
  </files>
  <read_first>
    .planning/phases/01-foundation-and-global-layout/01-RESEARCH.md (lines 269-288 for vite.config.js, lines 292-308 for main.css with @theme, lines 512-519 for main.js, lines 524-533 for package.json scripts)
    package.json (after Task 1 creates it via npm init)
  </read_first>
  <action>
    1. Run `npm init -y` in project root to create package.json.

    2. Run `npm install -D vite @tailwindcss/vite @tailwindcss/typography` to install build tools.

    3. Run `npm install alpinejs` to install Alpine.js as a production dependency.

    4. Run `npm install -D @fontsource/playfair-display @fontsource/raleway` for self-hosted Google Fonts (GDPR-friendly, no external requests).

    5. Update `package.json` scripts section to:
    ```json
    "scripts": {
      "dev": "vite",
      "build": "vite build",
      "preview": "vite preview"
    }
    ```
    Note: On Windows with Git Bash, `rm -rf dist` is not needed in the dev script since kirby-helpers detects dev mode by checking if Vite dev server is running, not by absence of dist/.

    6. Create `vite.config.js`:
    ```javascript
    import { defineConfig } from 'vite'
    import tailwindcss from '@tailwindcss/vite'

    export default defineConfig({
      plugins: [tailwindcss()],
      build: {
        manifest: true,
        outDir: 'dist',
        rollupOptions: {
          input: 'src/js/main.js',
        },
      },
      server: {
        port: 5173,
        strictPort: true,
      },
    })
    ```

    7. Create `src/css/main.css`:
    ```css
    @import "tailwindcss";
    @import "@fontsource/playfair-display/400.css";
    @import "@fontsource/playfair-display/700.css";
    @import "@fontsource/raleway/400.css";
    @import "@fontsource/raleway/600.css";
    @import "@fontsource/raleway/700.css";

    @theme {
      --color-primary: oklch(0.72 0.1 75);
      --color-primary-dark: oklch(0.62 0.1 75);
      --color-dark: oklch(0.15 0 0);
      --color-light: oklch(0.96 0.01 80);

      --font-heading: 'Playfair Display', serif;
      --font-body: 'Raleway', sans-serif;
    }
    ```

    8. Create `src/js/main.js`:
    ```javascript
    import Alpine from 'alpinejs'
    import '../css/main.css'

    window.Alpine = Alpine
    Alpine.start()
    ```
  </action>
