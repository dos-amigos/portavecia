  <action>
    Create `site/snippets/components/responsive-image.php` -- a reusable snippet that renders a `<picture>` element with WebP source and original format fallback.

    Parameters (passed via snippet() call):
    - `$image` (required) -- Kirby File object
    - `$preset` (optional, default `'default'`) -- srcset preset name from config (e.g., 'default', 'card', 'gallery')
    - `$sizes` (optional, default `'100vw'`) -- sizes attribute for responsive selection
    - `$alt` (optional) -- alt text, falls back to `$image->alt()->value()` then empty string
    - `$lazy` (optional, default `true`) -- if true, adds `loading="lazy" decoding="async"`; set false for hero/above-fold
    - `$class` (optional, default `''`) -- CSS class for the `<img>` element

    Implementation:
    ```php
    <?php
    /** @var \Kirby\Cms\File $image */
    $preset = $preset ?? 'default';
    $lazy   = $lazy ?? true;
    $alt    = $alt ?? $image->alt()->value() ?? '';
    $sizes  = $sizes ?? '100vw';
    $class  = $class ?? '';
    ?>
    <?php if ($image): ?>
    <picture>
      <source
        srcset="<?= $image->srcset($preset . '-webp') ?>"
        sizes="<?= $sizes ?>"
        type="image/webp"
      >
      <img
        src="<?= $image->resize(640)->url() ?>"
        srcset="<?= $image->srcset($preset) ?>"
        sizes="<?= $sizes ?>"
        alt="<?= esc($alt) ?>"
        width="<?= $image->width() ?>"
        height="<?= $image->height() ?>"
        <?php if ($lazy): ?>loading="lazy" decoding="async"<?php endif ?>
        <?php if ($class): ?>class="<?= $class ?>"<?php endif ?>
      >
    </picture>
    <?php endif ?>
    ```

    IMPORTANT: The WebP source preset name is constructed by appending `-webp` to the preset name. This matches the config presets defined in Plan 01 (default-webp, card-webp, gallery-webp).
  </action>
  <verify>
    <automated>test -f site/snippets/components/responsive-image.php && grep -c "<picture>" site/snippets/components/responsive-image.php && grep -c "srcset" site/snippets/components/responsive-image.php && grep -c "loading" site/snippets/components/responsive-image.php</automated>
  </verify>
